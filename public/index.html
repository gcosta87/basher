Basher, un pequeño servicio web escrito en Bash!

Basher trabaja con socat e intenta ser flexible para determinados contextos donde se requiere montar un servidor para mostrar información vía web o bien montar un servicio mínimo para testing y/o brindar información del entorno al ejecutar comandos de consola.
