Basher, un pequeño servicio web escrito en Bash!

Basher trabaja con socat para montar un pseudo-servidor web e intenta ser flexible para determinados contextos donde se requiere montar un servicio para mostrar información vía web o bien montar uno mínimo para testing y/o brindar información del entorno.
